线程，进程
```
进程：是才做系统资源分配的最小单位；
线程：是进行资源调度单位。
```
Java内存模型
```
线程共享：方法区，堆区
线程私有：虚拟机栈，本地方法栈，程序计数器
```
线程和进程的关系
```
对应关系：一个进程可以对应多个线程。
健壮性：进程和进程之间互相隔离，一个进程崩溃不会影响其他的进程
       一个线程崩溃可能导致整个进程崩溃
       进程的健壮性要比线程好
资源：创建进程意味着开辟内存空间，同一个进程下的线程共享部分空间
     开辟一个进程要比开辟一个线程占用的资源多
```
